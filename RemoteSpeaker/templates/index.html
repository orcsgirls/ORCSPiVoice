AIY Voice Kit

Controlling the Button Led

Link /led?value=color. Off will turn it off.

Speak text

Link /say?value=text_to_speak. Note the text needs to be 'htmlified' e.g. spaces are %20.

Sound files

Link /play?value=sound. The available sounds are in the Sounds directory of the server. They need to be in WAV format. Link /list lists available sounds.

Volume files

Link /volume?value=level. This will set the volume from 0 to 100%.
